TH3 Core Documentation
======================

This folder contains documentation for building, running, configuring, and operating TH3 Core.

Important files
---------------

- th3-conf.md - TH3 configuration file reference.
- init.md - service/init script setup notes.
- build-ubuntu.md - Ubuntu build notes.
- build-windows.md - Windows build notes.
- build-osx.md - macOS build notes.
- REST-interface.md - unauthenticated REST interface notes.
- JSON-RPC-interface.md - JSON-RPC interface notes.
- zmq.md - ZMQ interface notes.
- tor.md - Tor support notes.

Common commands
---------------

Start daemon:

    src/th3d -daemon

Use CLI:

    src/th3-cli getblockcount
    src/th3-cli getnetworkinfo
    src/th3-cli stop
